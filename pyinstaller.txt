First, run this code to change direction (change to your actual directory):
cd /Users/YourName/directory/PyWareFishing
Then, run this code to convert to app:

pyinstaller --onedir --windowed \
--icon="icon.icns" \
"PyWareFishingV3.py"

Finally, run this code to convert to App

hdiutil create -volname "VolumeName" -srcfolder cd /Users/YourName/directory/PyWareFishing
 -ov -format UDZO name.dmg
