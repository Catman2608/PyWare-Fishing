Step 1: run this code to change direction (change to your actual directory):
cd /Users/YourName/directory/PyWareFishing
Step 2: Convert to application

macOS command:
pyinstaller --onedir --windowed \
--icon="icon.icns" \
"PyWareFishingV4.py"

Windows command:
pyinstaller --onefile --windowed \
--icon="icon.ico" \
"PyWareFishingV4.py"

Step 3: Run this code to create setup instructions (if macOS) and if Windows compress to ZIP

hdiutil create -volname "VolumeName" -srcfolder cd /Users/YourName/directory/PyWareFishing
 -ov -format UDZO name.dmg

For me, the first line of code in the terminal looks like this:
cd "/Users/hoangphuc/Documents/Apps/Macros/PyWare Fishing V4"
Third line:
hdiutil create -volname "macOS.dmg" -srcfolder "/Users/hoangphuc/Documents/Apps/Macros/PyWare Fishing V4/dist" -ov -format UDZO macOS.dmg
